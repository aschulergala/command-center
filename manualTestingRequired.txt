This file tracks features that require manual testing with a real wallet.

## wallet-connection (Iteration 8)

### Test Requirements
Please test wallet connection with MetaMask: connect, reject, disconnect, and auto-reconnect flows.

### Checklist
- [ ] MetaMask popup appears when clicking Connect
- [ ] Approving connection shows address in UI
- [ ] Rejecting connection shows error message
- [ ] Disconnect clears UI and localStorage
- [ ] Page reload auto-reconnects if previously connected
- [ ] Test with MetaMask not installed (shows install prompt)

### How to Test
1. Start the dev server: `npm run dev`
2. Open http://localhost:5173 in a browser with MetaMask installed
3. Click "Connect Wallet" and test each scenario above
