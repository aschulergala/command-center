# Manual Testing Required

This file tracks features that require manual testing with a real wallet.

## wallet-connection (Iteration 8)

### Test Requirements
Please test wallet connection with MetaMask: connect, reject, disconnect, and auto-reconnect flows.

### Checklist
- [ ] MetaMask popup appears when clicking Connect
- [ ] Approving connection shows address in UI
- [ ] Rejecting connection shows error message
- [ ] Disconnect clears UI and localStorage
- [ ] Page reload auto-reconnects if previously connected
- [ ] Test with MetaMask not installed (shows install prompt)

### How to Test
1. Start the dev server: `npm run dev`
2. Open http://localhost:5173 in a browser with MetaMask installed
3. Click "Connect Wallet" and test each scenario above

## galachain-client (Iteration 9)

### Test Requirements
Please verify that write operations trigger MetaMask signing popup.

### Checklist
- [ ] Verify submit() triggers MetaMask signature popup
- [ ] Transfer operation prompts for wallet signature
- [ ] Mint operation prompts for wallet signature
- [ ] Burn operation prompts for wallet signature
- [ ] User can approve transaction - succeeds
- [ ] User can reject transaction - shows error message

### How to Test
1. Start the dev server: `npm run dev`
2. Connect your wallet using the Connect Wallet button
3. Once token operations are implemented, test transfer/mint/burn flows
4. Verify MetaMask popup appears for each write operation
